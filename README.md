# Minimapper::Extras

Extra tools for [minimapper](https://github.com/joakimk/minimapper).

## Installation

Add this line to your application's Gemfile:

    gem 'minimapper-extras'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minimapper-extras
